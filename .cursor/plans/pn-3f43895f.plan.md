# PNX Landing Page – Premium Visual & Motion Upgrade

## Scope

Enhance `src/pages/home/enhanced-home.tsx` to a cinematic dark theme with glowing gradients, smooth motion, and lightweight 3D/particle visuals. Preserve existing content structure and copy.

## Visual Direction

- Dark theme with deep navy base, electric blue/purple glowing accents
- Glassmorphic surfaces, faint borders, soft shadows
- Tailwind-based gradients + subtle glows; global smooth scrolling

## Tech Stack

- Tailwind CSS (existing)
- Framer Motion (existing) for entrance/scroll/parallax/magnetic hover
- React Three Fiber + Drei for lightweight 3D backgrounds
- Lottie fallback for mobile (for heavy scenes)

## Key Upgrades

### 1) Navbar (single glassmorphic bar)

- Consolidate to one sticky, translucent navbar with blur and faint shadow on scroll
- Links: Dashboard, Marketplace, Messages, Community, Pricing, About, Contact
- Right CTA group: Create Prompt, Log In, Sign Up with glowing hover/magnetic effect
- Smooth shrink on scroll

Implementation:

- Create `components/layout/PremiumNavbar.tsx` with scroll listener (Framer Motion `useScroll`) to animate height/shadow
- Replace local header in `enhanced-home.tsx` with `PremiumNavbar`

### 2) Hero – retain content, add background visuals

- Keep heading, subtext, and buttons
- Background Option A (default): smooth gradient waves + particle field reacting to cursor
- Background Option B (feature flag): AI Core sphere (glowing) with neural lines
- Floating light particles following cursor (very light)

Implementation:

- Add `components/visuals/HeroBackground.tsx`
- R3F Canvas with:
- Default: shader-driven gradient plane (simple fragment shader) + instanced particles
- Toggleable: low-poly sphere with bloom (Drei), rotating slowly
- ReduceMotion: disable heavy effects; show CSS gradient

### 3) Emotional Impact Section (new, below hero)

- Title: “Owning AI Tools ≠ Knowing AI.”
- Subtext and final line: “PNX: Unlock Your True Potential.”
- Scroll-driven reveal: AI key floats in → inserts into car → lights glow

Implementation:

- Add `components/visuals/ImpactScene.tsx`
- Desktop: R3F scene with two GLTF primitives (car silhouette, key), simple animation timeline via `useFrame` + Framer controls
- Mobile: Lottie animation fallback or gradient shimmer
- Insert non-intrusively below hero; preserve page’s existing order of other sections

### 4) Global Motion/Polish

- Section fade-up on view (Framer Motion)
- Subtle parallax using `useScroll`/`useTransform` for hero and key visuals
- Soft glowing borders for CTAs/feature cards
- Button hover pulse and magnetic hover effect (slight translate toward cursor)
- Typography: keep Inter; tighten tracking/leading and weight for hierarchy

### 5) Responsiveness & Performance

- Feature-detect reduced-motion; disable heavy 3D
- Mobile: swap to Lottie/gradient shimmer backgrounds
- Lazy-load Three.js bundles (dynamic import) below the fold
- Cap particle count; limit post-processing
- Guard with environment flag if needed
