# Webhook MCP Server

MCP server for webhook management and delivery.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running

```bash
python server.py
```

## Tools

- `webhook_create` - Create a new webhook
- `webhook_trigger` - Trigger a webhook
- `webhook_list` - List all webhooks
- `webhook_get` - Get webhook details
- `webhook_delete` - Delete a webhook
- `webhook_get_deliveries` - Get delivery history

## Port

This server runs on port **9012**.
