# Analytics MCP Server

MCP server for analytics, usage tracking, and metrics.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running

```bash
python server.py
```

## Tools

- `analytics_track_event` - Track an event
- `analytics_get_events` - Get events with filters
- `analytics_get_metrics` - Get aggregated metrics
- `analytics_increment_metric` - Increment a counter metric
- `analytics_set_metric` - Set a metric value
- `analytics_get_user_activity` - Get user activity summary

## Port

This server runs on port **9011**.
